RainGL
- Rain simulation using OpenGL & Cuda
- Compile
```
$ make
$ make run #Execute
$ make cudarun #Execute to get more fps
```

- Dependenqcies
+ Installl freeglut & SDL package
```
$ APPS="freeglut3 freeglut3-dbg freeglut3-dev libsdl1.2-dev libsdl-image1.2-dev "
$ sudo apt-get install $APPS
```
